# Quick Start: Backup & Restore

## 📤 Export Your Data (Backup)

1. Open **Settings** ⚙️
2. Scroll to **Data Management**
3. Tap **"Export All Data to JSON"** (green button)
4. Choose where to save:
   - **iCloud Drive** (recommended - sync across devices)
   - **On My iPhone** (local only)
   - Any folder you prefer
5. Done! ✅

**File name format**: `BudgetExpense_Backup_04-23-2026.json`

---

## 📥 Import Your Data (Restore)

1. Open **Settings** ⚙️
2. Scroll to **Data Management**
3. Tap **"Import Data from JSON"** (blue button)
4. Select your backup file
5. Tap **"Import"** in confirmation dialog
6. Done! ✅

⚠️ **Warning**: This replaces ALL your current data!

---

## 💡 Quick Tips

### When to Backup?
- ✅ Before updating the app
- ✅ Weekly (set a reminder!)
- ✅ Before switching devices
- ✅ After adding lots of new data

### Where to Save Backups?
- 🌟 **Best**: iCloud Drive (automatic sync, accessible anywhere)
- 👍 **Good**: Google Drive, Dropbox (via Files app)
- ⚠️ **OK**: Local storage (remember to backup your phone!)

### How Many Backups?
Keep at least **3 recent backups**:
- Current week
- Last week
- Last month

## 🆘 Quick Troubleshooting

### Export doesn't work?
- Check storage space
- Try saving to iCloud Drive instead
- Check app permissions

### Import doesn't work?
- Make sure it's a `.json` file
- Verify the file isn't corrupted
- Try copying to iCloud Drive first

### Can't find my backup?
- Open **Files app** → **Browse**
- Check **iCloud Drive** or **On My iPhone**
- Search for "BudgetExpense"

---

## 📱 Switching to New iPhone?

### Option 1: Via iCloud Drive (Easiest)
1. **Old iPhone**: Export to iCloud Drive
2. **New iPhone**: Install app, import from iCloud Drive
3. Done! 🎉

### Option 2: Via AirDrop
1. **Old iPhone**: Export → Share → AirDrop to new device
2. **New iPhone**: Open file, tap "Import"
3. Done! 🎉

### Option 3: Via Email/Messages
1. **Old iPhone**: Export → Share → Email/Message to yourself
2. **New iPhone**: Download attachment, import
3. Done! 🎉

## 🔒 Security Tips

- ✅ Backups are **NOT encrypted** by default
- ✅ Use **iCloud Drive** for automatic encryption
- ✅ Don't share backups publicly (they contain your financial data!)
- ✅ Keep backups in secure locations only

---

## ❓ FAQ

**Q: Will this backup my PIN?**  
A: No, security settings are stored separately for safety.

**Q: Can I use this to transfer data between devices?**  
A: Yes! Export on one device, import on another.

**Q: What if I have both CSV and JSON backups?**  
A: JSON is complete backup (can restore), CSV is for viewing only.

**Q: How big is the backup file?**  
A: Usually a few KB to a few MB, depending on data amount.

**Q: Can I edit the JSON file?**  
A: Yes, but be careful! Invalid JSON won't import.
